# 用let不用var

ES6之前我们用var声明一个变量，但是它有很多弊病：

  • 因为没有块级作用域，很容易声明全局变量
  • 变量提升
  • 可以重复声明 还记得这道面试题吗？
var a = [];
for (var i = 0; i < 10; i++) {
  a[i] = function () {
    console.log(i);
  };
}
a[6](); // 10
a[7](); // 10
a[8](); // 10
a[9](); // 10
